# 📊 Retail Sales and Profit Dashboard

## 📌 Overview
An interactive Power BI dashboard to analyze retail sales and profit performance across categories,
regions, and customers.

## 🎯 Key Insights
- Total Sales & Profit tracking  
- Sales by Category & Sub-category  
- Region-wise performance analysis  
- Monthly sales trends  
- Top products performance  

## 🛠 Tools Used
- Power BI  
- Power Query  
- DAX  
- Excel Dataset  

## 🚀 Features
- Interactive filters & slicers  
- KPI cards for quick insights  
- Trend and comparison visuals  

## 📁 Project Files
- Retail Sales and Profit Dashboard.pbix file (Power BI dashboard)  
- Dataset file  

## ⭐ Purpose
To demonstrate data analysis and visualization skills for business decision-making.
